# Overview

This repository contains a simple GUI calculator written in Python using the Tkinter toolkit. The application provides a fixed-size window with a numeric display and a 4x5 button grid that supports the following operations:

- Numeric input (0–9) and decimal point
- Basic arithmetic: addition, subtraction, multiplication, division
- Percentage, delete (backspace), clear (AC)
- Square root and evaluation (`=`)

The UI demonstrates layout management with `Frame` and `grid`, dynamic display scaling to avoid overflow, and an event-driven command pattern for buttons. The goal is to explore practical GUI concerns (layout, input handling, display formatting) while producing a compact, demo-ready tool.
